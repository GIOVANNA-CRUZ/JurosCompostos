<h1>JurosCompostos</h1>

<h2>Description and Funcionalities</h2>
This project has the objective of calculating how long a person would become a millionaire in the compound interest system. There is a first page which
is the index, where the amount of the contribution and the interest for the investment are requested. The second page is the answer page, where the 
Compound Interest class methods (found in the "data" package) are called, which performs this calculation.This project was made exclusively in HTML, 
CSS and Java.

<h2>How to use</h2>
To open this project on your machine it is necessary to have the Java SDK installed and some IDE, such as Eclipse, IntelliJ IDEA or NetBeans. 
Download it, configure it on the machine and open the files.
